# Notes

Pricing information changes frequently across AI vendors.

The audit engine intentionally uses simplified plan assumptions suitable for MVP-level recommendation logic rather than attempting to model every enterprise contract variation, negotiated enterprise contract, or token-level API billing edge case.

User-entered monthly spend values are treated as the source of truth because many startups operate with discounts, credits, hybrid API usage, or non-standard billing arrangements that differ from public retail pricing.
